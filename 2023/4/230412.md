## **`오늘 배운 내용`**

- ### 객체지향 프로그래밍 특징
  - 상속
    - 부모클래스로 부터 상속받은 자식 클래스는 부모클래스의 메서드와 변수 사용 가능
  - 추상화
    - 추상 클래스는 선언 불가능
  - 캡슐화
    - public
      - 모두 접근 가능 
    - protected
      - 같은 패키지 내 접근 가능
    - private
      - 같은 클래스만 접근 가능 
      - 정보 은닉의 기능
      - getter / setter 메서드를 이용해 private 변수 설정
  - 다형성
    - 오버로딩
      - 메서드의 이름은 같고 매개변수의 갯수나 타입이 다른 함수를 정의하는 것을 의미
    - 오버라이딩
      - 상위 클래스의 메서드를 하위 클래스가 재정의 하는 것
